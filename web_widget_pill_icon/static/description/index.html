Flexible Style Variations

Mix and match base and semantic classes to create the perfect look. Supports solid colors, outlines, and different sizes (sm, lg) to fit any UI requirement.

Available Display Options:

  • Solid Semantic Pills
  • Transparent Outline Style
  • Size Modifiers (Small & Large)

Built-in Semantic Styles Reference

Option Value Mapping Standard Style Outline Style
fa-check-circle success Success Success
fa-warning warning Warning Warning
fa-trash danger Danger Danger
fa-star primary Primary Primary
fa-minus-circle muted Muted Muted

XML Configuration

<field name="status"
       widget="pill_icon"
       options="{
           'base': 'pill outline sm',
           'values': {
               'done': 'fa-check success',
               'error': 'fa-exclamation danger'
           }
       }"/>
