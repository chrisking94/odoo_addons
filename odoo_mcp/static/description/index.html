🎯 What Does It Do?

This addon turns your Odoo instance into a Model Context Protocol (MCP) Server, enabling AI agents and Large Language Models to interact with your Odoo data seamlessly.

  • Expose any Odoo model method to AI agents
  • Simple @mcp_tool decorator - just one line of code
  • Automatic JSON schema generation from Python type hints
  • Built-in SSE endpoint for real-time communication
  • Full JSON-RPC 2.0 support
  • Zero configuration required

⚡ Super Simple Usage Example

Step 1: Add the decorator to your model method

from odoo import models from odoo_mcp.decorators import mcp_tool class ResPartner(models.Model): _inherit = 'res.partner' @mcp_tool(description="Search for customers by name") def search_customers(self, name: str): """This method is now accessible to AI agents!""" customers = self.search([('name', 'ilike', name)]) return [{ 'id': c.id, 'name': c.name, 'email': c.email } for c in customers]

Step 2: That's it! Your method is now available at /mcp/sse

Step 3: AI agents can now call your method:

# AI Agent sends this JSON-RPC request: { "jsonrpc": "2.0", "id": 1, "method": "tools/call", "params": { "name": "res.partner.search_customers", "arguments": {"name": "John"} } } # Gets this response: [ {"id": 12, "name": "John Smith", "email": "john@example.com"}, {"id": 45, "name": "John Doe", "email": "johnd@example.com"} ]
